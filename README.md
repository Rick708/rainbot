# README

降水確率通知LineBot


## 説明
当日の大阪府エリアの降水確率を取得し（6-12時、12-18時、18-24時）のどれか１つが50%以上であった場合にメッセージを送信しています。
大阪府エリア意外にも全都道府県指定することもでき、通知のラインの50％も変動可能にしており
余計な通知が来る事を避ける為50％以下は通知が来ない様にしております。「今日、明日、明後日」をコメントすると降水確率を教えてくれる
機能もあります。


## 開発環境
Line API
rails
heroku
